# Birthday Wishes - 精致版生日祝福网页

这是一个采用“轻法式奶油风”设计的精致生日祝福网页。相比初始版本，此版本大幅强化了视觉动效（炫技式堆砌）、玻璃拟态（Glassmorphism）UI、以及更细腻的交互体验。

## ✨ 核心亮点

1.  **动态粒子背景**：Canvas 实时渲染的漂浮星光，让整个画面充满呼吸感。
2.  **玻璃拟态 UI**：极致的毛玻璃效果，搭配淡金与柔粉配色，治愈且高级。
3.  **炫技式动效**：
    *   **3D 蜡烛交互**：模拟真实长按“吹灭”动效。
    *   **3D 礼盒拆解**：丝滑的盒盖分离动画。
    *   **打字机祝福**：长篇祝福逐字浮现，增加仪式感。
    *   **全场彩纸雨**：解锁与达成目标时的即时反馈。
4.  **功能细化**：包含解锁序曲、互动抽签、动态祝福、倒计时纪念、壁纸生成及随机悄悄话。

## 🛠️ 素材准备清单 (需自行放入 assets/audio/)

为了达到最佳效果，请准备以下素材并放入对应文件夹（确保文件名完全一致）：

| 文件名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `bgm.mp3` | 背景音乐 | 建议选择温馨、轻快、循环性强的纯音乐。 |
| `cheer.mp3` | 音效 | 用于解锁成功时的欢呼或撒花声。 |
| `blow.mp3` | 音效 | 吹灭蜡烛时的声音。 |
| `pop.mp3` | 音效 | 抽签卡片翻转或按钮点击声。 |
| `gift_open.mp3` | 音效 | 礼盒打开时的音效。 |
| `piano.mp3` | 音效 | 输入密码时的轻微按键声（已内置逻辑）。 |
| `voice.mp3` | 语音 | 你的专属录音祝福（可选）。 |

## ⚙️ 定制指南

打开 `js/script.js`，修改顶部的 `config` 对象：

```javascript
const config = {
    nickname: "小仙女",       // 对方昵称
    birthday: "03-06",        // 生日日期 (MM-DD)
    unlockPassword: "生日快乐", // 解锁口令
    fortunes: [...],          // 随机签文池
    shortWish: "...",         // 短标题
    longWish: "...",          // 长篇祝福 (支持 \n 换行)
    secrets: [...]            // 随机悄悄话池
};
```

## 🚀 运行与调试

*   **本地查看**：建议使用 VS Code 的 `Live Server` 插件运行，以保证音频和 Canvas 正常工作。
*   **移动端**：按下 `F12` 并切换到“手机模式”，本项目已针对手机竖屏进行深度适配。

## 🎨 视觉微调

如需调整主色调，请修改 `css/style.css` 中的 `:root` 变量：
*   `--cream-bg`: 背景奶油色
*   `--soft-pink`: 柔粉色
*   `--pale-gold`: 标志性淡金色
